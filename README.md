# Webpack Boiler w/ Babel

## Installation

Install dependencies (Webpack, Babel)

```sh
$ npm install
```

## Compile

To compile all js to dist/app.bundle.js

```sh
$ npm run webpack
```
To watch run
```sh
$ npm run webpack -w
```

## Run
Install live-server globally
```sh
$ npm install live-server -g
```

```sh
$ npm start
```
